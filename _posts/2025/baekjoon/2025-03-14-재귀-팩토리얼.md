# 재귀문(팩토리얼)

## 팩토리얼

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

- 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

- 출력
첫째 줄에 N!을 출력한다.


- 예제1 입력
10

- 예제1 출력
3628800

### 논리적 사고


### 풀이01. 재귀 - for문으로 푸는법
```python
N = int(input())

result = 1

for i in range(N):
    result *= (i+1)

print(result) 
```

### 풀이02. 재귀 - 함수로 풀기
```python
def factorial(n):
    if n < 2:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(int(input())))
```

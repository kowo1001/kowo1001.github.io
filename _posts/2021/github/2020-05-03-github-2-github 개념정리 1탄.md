## GitHub란?
- GitHub는 친구, 동료는 물론 낯선 사람과 함께 코드를 공유하고자 만든 Git 리포지토리의 호스팅 서비스입니다. 

## 일반적인 Git 리포지토리 호스팅 서비스가 아니다!
- GitHub는 Git 리포지토리(repository) 호스팅 기능 이외에도 개발자와 팀이 빠른 속도로 좋은 품질의 코드를 만들어 낼 수 있도록 하는 기능을 제공합니다.

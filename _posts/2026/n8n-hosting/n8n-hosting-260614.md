# 스킬이 항상 지키는 안전 규칙

사용자가 몰라도 됩니다 -- 이 규칙들이 자동으로  사고를 막아줍니다. 

### 암호화 키는 따로, 화면에 출력 금지

설치 폴더 밖 비밀번호 관리자에 보관

### 데이터 삭제 명령 함부로 실행 안함

기존 설정은 읽기 전엔 덮어쓰지 않음

### 인터넷 공개는 확인 후에만

Funnel(공개)은 사용자 동의 없이 켜지 않음

### 위험한 작업 전 항상 백업

업그레이드, 설정 변경 전 자동으로 챙김

### 자동 업데이트 금지

되돌릴 수 없는 작업은 사람 확인 후에만

### 내부 부품은 외부에 노출 안함

데이터 창고는 바깥으로 열어 두지 않음

# 용어 정리

### Tailscale

내 기기들끼리만 연결되는 "사설 안전망(VPN)". 복잡한 설정 없이 내 기기를 한 울타리로 묶음.

### Tailscale Serve

내 tailnet 안에서만 HTTPS로 n8n에 접속. 인터넷 전체엔 공개되지 않아 안전 (원격 추천)

### Ollama / 외부 LLM API 

Ollama = 내 컴퓨터에서 AI 직접 실행. 외부 LLM API = OpenAI, Claude 등
클라우드 AI를 키로 호출.

### Qdrant / 벡터 DB 

AI가 문서를 "의미로 검색"하게 해주는 특수 창고. RAG(내 문서 기반 답변)에 쓰임.

### 웹훅(webhook)

"어떤 일이 생기면 이 주소로 알려줘" 라고 외부 서비스에 등록하는 콜백 주소.

https://github.com/jeongkpa/n8n-self-hosting-onboard >>> 이 스킬을 다운 받아서, 최신으로 업데이트를 하고 이 스킬 기반으로 n8n 서비스를 셀프호스팅을 해보자

### 설치는 쉽게, 운영은 안전하게

대화로 설치하고, "백업 먼저" 원칙으로 오래 굴립니다.
처음 선택은 나중에 얼마든지 바꿀 수 있습니다.
